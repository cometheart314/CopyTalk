ClipVoice

macOS 用テキスト読み上げメニューバーアプリ

ClipVoice について

ClipVoice は、クリップボードのテキストやアプリケーションで選択したテキストを音声で読み上げる macOS メニューバーアプリです。設定不要ですぐに使い始められます。

GitHub でソースコードを見る

主な機能

すぐに使える

Apple 標準の音声合成を搭載。API キーの設定なしで、インストール直後から読み上げが可能です。

高品質音声

Google Cloud Text-to-Speech の Neural2 音声にも対応。API キーを設定するだけで、より自然な音声にアップグレードできます。設定方法

⌘C ⌘C で読み上げ

テキストを選択して ⌘C を素早く2回押すだけで読み上げ開始。どのアプリからでも使えます。

自動言語検出

日本語と英語を自動的に判別し、適切な音声で読み上げます。テキスト全体で一貫した音声を使用します。

段落ポーズ

段落の区切りで自然な間を入れて読み上げます。長い文章でも聞き取りやすい読み上げを実現。

サービスメニュー対応

macOS のサービスメニューに統合。他のアプリケーションから選択テキストを直接読み上げできます。
