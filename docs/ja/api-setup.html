Google Cloud API キーの取得方法

高品質な Neural2 音声を使うための設定ガイド

ClipVoice は API キーなしでも Apple 標準音声で読み上げができますが、Google Cloud Text-to-Speech の API キーを設定すると、より自然で高品質な Neural2 音声が使えるようになります。

無料枠について：Google Cloud Text-to-Speech には毎月の無料枠があります。Neural2 音声は月に100万文字まで無料で利用できます（2026年4月現在）。個人利用であれば無料枠内で収まることがほとんどです。

手順

1Google アカウントでログイン

Google Cloud Console にアクセスし、Google アカウントでログインします。

Google Cloud Console を開く

2プロジェクトを作成

  1. 画面上部のプロジェクト選択ドロップダウンをクリック
  2. 「新しいプロジェクト」をクリック
  3. プロジェクト名に任意の名前を入力（例: ClipVoice）
  4. 「作成」をクリック
  5. 作成したプロジェクトが選択されていることを確認
初めて Google Cloud を使う場合、利用規約への同意を求められることがあります。

3請求先アカウントを設定

API を使用するには請求先アカウントの設定が必要です（無料枠内であれば課金されません）。

  1. 左メニューの「お支払い」をクリック
  2. 「請求先アカウントをリンク」をクリック
  3. クレジットカード情報を入力して設定を完了
注意：無料枠を超えた場合のみ課金されます。心配な場合は、後述の「予算アラートの設定」で上限を設定することをおすすめします。

4Text-to-Speech API を有効化

  1. 以下のリンクから Text-to-Speech API のページを開きます
  2. 「有効にする」ボタンをクリック
Text-to-Speech API ページを開く

5API キーを作成

  1. 以下のリンクから「認証情報」ページを開きます
  2. 画面上部の「+ 認証情報を作成」をクリック
  3. 「API キー」を選択
  4. API キーが生成されます。表示されたキーをコピーしてください
認証情報ページを開く

6API キーを制限（推奨）

セキュリティのため、API キーの使用を Text-to-Speech API のみに制限することをおすすめします。

  1. 作成した API キーの名前をクリック
  2. 「API の制限」セクションで「キーを制限」を選択
  3. ドロップダウンから「Cloud Text-to-Speech API」を選択
  4. 「保存」をクリック

7ClipVoice に API キーを設定

  1. ClipVoice のメニューバーアイコンをクリック
  2. 「Settings...」を選択
  3. 「API Key」フィールドにコピーした API キーを貼り付け
  4. ステータスが「Using Google Cloud Text-to-Speech」に変わったら設定完了です
  5. 「Test」ボタンで音声を確認できます

予算アラートの設定（おすすめ）

想定外の課金を防ぐため、予算アラートを設定しておくと安心です。

  1. Google Cloud Console の「お支払い」→「予算とアラート」を開く
  2. 「予算を作成」をクリック
  3. 月額の予算額を設定（例: 100円）
  4. アラートのしきい値を設定（50%、90%、100% など）
予算アラートは通知のみで、自動的に API を停止するものではありません。しかし、無料枠を超えそうな場合に事前に知ることができます。

トラブルシューティング

「API key not valid」エラーが出る

  • API キーが正しくコピーされているか確認してください
  • Text-to-Speech API が有効になっているか確認してください
  • 請求先アカウントがプロジェクトにリンクされているか確認してください

「Permission denied」エラーが出る

  • API キーの制限が正しく設定されているか確認してください
  • 「Cloud Text-to-Speech API」が許可リストに含まれているか確認してください

音声が再生されない

  • macOS のサウンド出力設定を確認してください
  • Settings の「Test」ボタンでエラーメッセージを確認してください
