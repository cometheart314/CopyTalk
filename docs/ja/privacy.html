CopyTalk のプライバシーに関する方針

概要

CopyTalk はユーザーのプライバシーを尊重します。本アプリは、個人情報の収集、保存、または第三者への送信を行いません。

データの収集について

CopyTalk は以下のデータを収集しません：

  • 個人を特定できる情報（氏名、メールアドレス等）
  • 位置情報
  • 連絡先やカレンダーのデータ
  • クリップボードや読み上げテキストの内容

ネットワーク通信

CopyTalk は以下の場合にネットワーク通信を行います：

  • Google Cloud Text-to-Speech API を使用する場合、読み上げテキストが Google のサーバーに送信されます
  • この通信はユーザーが API キーを設定した場合にのみ発生します
  • API キーを設定しない場合（Apple 標準音声の使用時）、ネットワーク通信は一切行いません
  • Google Cloud のプライバシーポリシーについては Google Cloud Privacy Notice をご参照ください

ローカルデータ

アプリの設定（音声の選択、速度設定等）はお使いの Mac のローカルストレージに保存されます。Google Cloud API キーは macOS のキーチェーンに安全に保存され、外部に送信されることはありません。

お問い合わせ

プライバシーに関するご質問は、GitHub Issues までお願いします。
