How to Get a Google Cloud API Key

Setup guide for high-quality Neural2 voices

ClipVoice works out of the box with Apple's built-in voices, but by setting up a Google Cloud Text-to-Speech API key, you can upgrade to more natural-sounding Neural2 voices.

Free tier: Google Cloud Text-to-Speech includes a monthly free tier. Neural2 voices are free for up to 1 million characters per month (as of April 2026). Personal use typically stays well within this limit.

Steps

1Sign in to Google Cloud

Open the Google Cloud Console and sign in with your Google account.

Open Google Cloud Console

2Create a Project

  1. Click the project selector dropdown at the top of the page
  2. Click "New Project"
  3. Enter a project name (e.g., ClipVoice)
  4. Click "Create"
  5. Make sure the newly created project is selected
If this is your first time using Google Cloud, you may be asked to agree to the terms of service.

3Set Up a Billing Account

A billing account is required to use APIs (you won't be charged if you stay within the free tier).

  1. Click "Billing" in the left menu
  2. Click "Link a billing account"
  3. Enter your credit card information to complete the setup
Note: You will only be charged if you exceed the free tier. If concerned, see "Setting Up Budget Alerts" below to set a spending limit.

4Enable the Text-to-Speech API

  1. Open the Text-to-Speech API page using the link below
  2. Click the "Enable" button
Open Text-to-Speech API page

5Create an API Key

  1. Open the Credentials page using the link below
  2. Click "+ Create Credentials" at the top
  3. Select "API key"
  4. An API key will be generated. Copy the displayed key
Open Credentials page

6Restrict the API Key (Recommended)

For security, it's recommended to restrict your API key to only the Text-to-Speech API.

  1. Click on the name of the API key you just created
  2. Under "API restrictions", select "Restrict key"
  3. Select "Cloud Text-to-Speech API" from the dropdown
  4. Click "Save"

7Set the API Key in ClipVoice

  1. Click the ClipVoice icon in the menu bar
  2. Select "Settings..."
  3. Paste the copied API key into the "API Key" field
  4. The status should change to "Using Google Cloud Text-to-Speech"
  5. Use the "Test" buttons to verify the voice

Setting Up Budget Alerts (Recommended)

To prevent unexpected charges, it's a good idea to set up budget alerts.

  1. In Google Cloud Console, go to "Billing" → "Budgets & alerts"
  2. Click "Create budget"
  3. Set a monthly budget amount (e.g., $1)
  4. Configure alert thresholds (50%, 90%, 100%, etc.)
Budget alerts are notifications only and do not automatically stop API usage. However, they help you know in advance if you're approaching the free tier limit.

Troubleshooting

"API key not valid" error

  • Verify the API key was copied correctly
  • Check that the Text-to-Speech API is enabled
  • Ensure a billing account is linked to your project

"Permission denied" error

  • Check that the API key restrictions are configured correctly
  • Ensure "Cloud Text-to-Speech API" is in the allowed list

No audio playback

  • Check your macOS sound output settings
  • Use the "Test" button in Settings to check for error messages
