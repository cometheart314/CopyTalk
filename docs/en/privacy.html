Privacy Policy

ClipVoice's commitment to your privacy

Overview

ClipVoice respects your privacy. This application does not collect, store, or transmit any personal information.

Data Collection

ClipVoice does not collect:

  • Personally identifiable information (name, email address, etc.)
  • Location data
  • Contacts or calendar data
  • Clipboard or spoken text contents

Network Communication

ClipVoice communicates over the network in the following cases:

  • When using Google Cloud Text-to-Speech API, the text to be read aloud is sent to Google's servers
  • This communication only occurs when the user has configured an API key
  • When no API key is configured (using Apple's built-in voices), no network communication takes place
  • For Google Cloud's privacy practices, please refer to the Google Cloud Privacy Notice

Local Data

Application preferences (voice selection, speed settings, etc.) are stored on your Mac's local storage. The Google Cloud API key is securely stored in the macOS Keychain and is never transmitted externally.

Contact

For privacy-related inquiries, please visit GitHub Issues.
