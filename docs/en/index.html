ClipVoice

Text-to-Speech menu bar app for macOS

About ClipVoice

ClipVoice is a macOS menu bar app that reads clipboard text or selected text from any application aloud. It works right out of the box with no configuration required.

View Source on GitHub

Key Features

Works Instantly

Built-in Apple text-to-speech engine. Start reading text aloud immediately after installation with no API key required.

High-Quality Voices

Supports Google Cloud Text-to-Speech Neural2 and Chirp 3: HD voices. Simply set an API key to upgrade to more natural-sounding voices. Setup guide

⌘C ⌘C to Speak

Select text and quickly press ⌘C twice to start reading aloud. Works from any application.

Automatic Language Detection

Automatically detects Japanese and English, reading with the appropriate voice. Uses a consistent voice throughout the entire text.

Paragraph Pauses

Inserts natural pauses between paragraphs for comfortable listening, even with long texts.

Services Menu

Integrated with the macOS Services menu. Read selected text directly from other applications.
